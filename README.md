# sample-java-(Gradle)project

## Devolopment

> For Windows you might have to run all `./gradlew` commands
> as `./gradlew.bat` instead

### Building and Running

#### Build Project

````shell
./gradlew build
````

#### Running Project
Go to `build/lists` folder

````shell
java -jar sample-java-project-1.0-SNAPSHOT.jar
````

### Testing
#### Run Tests  

````shell
./gradlew test

````
